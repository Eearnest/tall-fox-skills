# Telegram Group Digest Skill

Summarizes the last 24 hours of messages in a Telegram group into a structured, scannable digest — covering announcements, top discussions, links shared, action items, and most active members.

## Hard rules

- This skill is **read-only** — it never sends, deletes, or modifies any messages
- Never expose private messages, personal data, or sensitive conversations in output
- Always state the time period covered and total message count
- Filter out spam, one-word replies, greetings, and low-value messages
- Never fabricate messages or activity — only summarize what actually exists
- If no messages found in the last 24 hours, say so clearly
- Respect group privacy — do not share digest output outside the group context

## When to Use

- User wants a summary of what happened in a Telegram group
- User missed messages and wants to catch up quickly
- Community manager wants a daily digest of group activity
- User wants to know the most important announcements from the last 24 hours
- User types /digest, /catchup, /recap, or /summary in the group
- User asks "what did I miss" or "summarize the group"

## Core Knowledge

### Key Principles

1. **Time-bound filtering** — Only analyze messages from the last 24 hours unless the user specifies a different window
2. **Signal over noise** — Ignore greetings, emoji-only messages, spam, and one-word replies
3. **Categorization** — Group messages by type: announcements, discussions, links, action items
4. **Member recognition** — Identify and credit most active contributors by username
5. **Action extraction** — Pull out tasks, follow-ups, deadlines, and commitments explicitly mentioned
6. **Link tracking** — Catalog every URL shared with a one-line description of what it links to
7. **Privacy first** — Never expose sensitive personal information or private conversations
8. **Context transparency** — Always state the time period and total message count at the top

### Message Categories

| Category | What to Look For |
|----------|-----------------|
| Announcements | Official updates, important news, pinned messages, policy changes |
| Discussions | Topic threads, debates, questions with multiple replies, decisions made |
| Links | URLs, articles, tools, resources, documentation, videos |
| Action Items | Tasks, commitments, deadlines, follow-ups, assignments |
| Events | Upcoming meetings, AMAs, workshops, deadlines |

### Filtering Rules

**Include:**
- Messages with substance — questions, answers, announcements, links, decisions
- Messages that generated replies or reactions
- Messages from admins or pinned content
- Any message containing a URL or resource

**Exclude:**
- Single emoji messages
- Pure greetings ("hi", "hello", "gm", "good morning")
- Duplicate or forwarded spam
- Off-topic one-liners with no replies
- Stickers and GIFs with no accompanying text

### Time Window Options

If user specifies a different window, honor it:
- "last hour" → 1 hour window
- "last 24 hours" → default 24 hour window
- "last week" → 7 day window
- "today" → from midnight to now
- "yesterday" → previous full day

### Mistakes to Avoid

- Never summarize private direct messages — only group messages
- Don't invent activity that didn't happen
- Don't include usernames in sensitive or controversial contexts
- Don't post the digest publicly unless the user explicitly asks
- Never share the digest outside the group it was generated from

## Guidelines

- Default time window is always 24 hours unless specified
- If the group has fewer than 10 messages in the window, note it's a quiet period
- For very active groups (500+ messages), focus on highest-engagement threads only
- Always end the digest with: "Generated by LegionPulse 🤖"
- If Telegram tool is not connected, ask the user to enable it in IronClaw Extensions
- Offer to schedule automatic daily digests after the first successful run
